Exercise :

**Project:** Build a basic "Portfolio Management Dashboard" application.

 

**Requirements:**

 

1. **User Authentication:**

    

    - Implement a basic authentication system (e.g., login/logout) using JWT (JSON Web Tokens).

2. **Portfolio Overview:**

    

    - Create a dashboard where users can view their investment portfolios.

    - Display a summary of assets, such as stocks, bonds, and mutual funds, including their current value, purchase price, and performance metrics.

3. **Transaction History:**

    

    - Implement a feature where users can view a history of their transactions (buy/sell) 

4. **Add/Edit Investments:**

    

    - Allow users to add new investments and edit existing ones.

5. **Technology Stack:**

    

    - **Front-end:** Use a modern JavaScript framework/library (React, Angular, or Vue.js).

    - **Back-end:** Use Node.js and typescript.

    - **Database:** Use whatever you want (redis, nosql, relational db)

    - **Version Control:** Use Git for source control and demonstrate commit history.

6. **Deliverable:**

    

    - The deliverable should be Dockerfile (or compose) which could run the application.
